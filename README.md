Lista de Exercícios II – Sistemas Distribuídos e Mobile
Centro Universitário UNA
Professor: Daniel Henrique Matos de Paiva

## Descrição

Este repositório contém a resolução da **Lista de Exercícios II**, que consiste na criação de estruturas **JSON** e suas respectivas **classes em C# (.NET)**.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

* **json/** → Contém os arquivos JSON criados para cada exercício.
* **models/** → Contém as classes C# correspondentes a cada estrutura JSON.

## Exercícios Implementados

1. Estrutura JSON para peças de um depósito de ferramentas.
2. Estrutura JSON para veículos de uma loja de automóveis.
3. Estrutura JSON para livros.
4. Estrutura JSON para informações de uma universidade.
5. Estrutura JSON para informações de um shopping.

Cada exercício possui:

* Um arquivo JSON com dados de exemplo.
* Uma classe C# representando a estrutura desses dados.
